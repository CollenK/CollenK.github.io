Photo Management Web App


Description

This Rails 4.2 app allows users to upload and manage photos to the cloud. It offers two distinct plans and uses custom Devise functionality and SendGrid for user sign up and confirmation in production.

Payment is handled via the Stripe API with a custom payment form. Cloud storage is provided using Amazon Web Services S3.

Stack

  • Ruby & Ruby on Rails (4.2)
  • JavaScript & jQuery
  • HTML5, CSS3 & Bootstrap
  • Stripe API
  • Devise
  • AWS S3
  • Git, GitHub, Heroku, SQLite & PostgreSQL
