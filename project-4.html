Blog App


Description

This basic Blog app is built on Rails 4.2, styled with Bootstrap and most features are hand-coded instead of relying on gems or Rails generators and scaffolds.

Features include an authentication and authorization system built from scratch with standard user and admin roles. Standard users can log in, view articles by category, view all users and add or delete their own articles with a title, content and category.

Admin users can edit and delete all users, articles, or categories.

Article views are organized using pagination and user profiles are displayed with an image via Gravatar.

Stack

  • Ruby & Ruby on Rails (4.2)
  • JavaScript & jQuery
  • HTML5, CSS3 & Bootstrap
  • Stripe API
  • Git, GitHub, Heroku, SQLite & PostgreSQL
