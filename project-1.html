Fuji Fanatics, an eCommerce Web App


Description

A Rails eCommerce web app built as part of the CareerFoundry Web Development course. The app was originally built on Rails 4.2, and upgraded to Rails 5.

Main features include user authentication and authorization (log in/log out/sign up), and an admin user that can manage all aspects of the app. Automated mailers on successful sign up and order placement are set up using ActionMailer.

Users can rate and comment on products and real-time updates are implemented using ActionCable..

Stripe checkout is implemented. The app has a test suite created with RSpec.

Stack

  • Ruby & Ruby on Rails (4.2 & 5)
  • JavaScript, jQuery & AJAX
  • HTML5, CSS3 & Bootstrap
  • Stripe API
  • Memcached, Redis & Dalli
  • Devise & CanCanCan
  • RSpec, Capybara & FactoryGirl
  • Git, GitHub, Heroku, New Relic, SQLite & PostgreSQL
