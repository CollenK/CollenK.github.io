eCommerce Web App

Description

A Rails eCommerce app built as part of the CareerFoundry Web Development course. The app was originally built on Rails 4.2, and upgraded to Rails 5.

Main features include user authentication (log in/log out/sign up), and admin user that can manage all aspects of the app. Automated mailers on successful sign up and order placement are set up using ActionMailer.

Users can rate and comment on products and Stripe checkout is implemented.

Stack

  • Ruby on Rails
  • JavaScript & jQuery
  • HTML5 & CSS3
  • Bootstrap
  • Stripe API
  • Memcached & Redis
  • Devise & CanCanCan
  • RSpec & Capybara

View this project on GitHub and live on Heroku.
